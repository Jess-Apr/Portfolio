프로젝트 소개

사용자가 지정한 웹 페이지의 테이블을 감지하여 동적 크롤링 후, 데이터를 엑셀 파일로 다운로드할 수 있게 해주는 크롬 확장 프로그램

  • 진행 기간: 2025.01 ~ 진행중
  • 진행 인원: 프론트엔드 1인, 백엔드 1인
  • 담당 업무: 프론트엔드 개발

사용한 기술 스택

TypeScriptReactJotaiTailwind CSSJest

주요 작업 내용

  • 클라이언트-서버 암호화 구현

    • 개인정보가 포함된 데이터 필드를 AES 알고리즘으로 암호화한 뒤, AES 키를 서버의 RSA Public Key로 암호화하여 요청 헤더에 포함해 전송

    • 응답 데이터 암호화가 필요한 경우, 클라이언트에서 RSA 키를 생성하고 요청 헤더에 Public Key를 포함하여 전송

    • 암·복호화 로직을 Axios Interceptor에 구현하여 모든 요청과 응답에서 자동으로 암·복호화가 수행되도록 설정

  • 브라우저 동작 제어 로직 구현

    • Chrome Extension의 Content Script를 활용하여 사용자가 크롤링할 테이블을 직접 선택할 수 있도록 기능 제공

  • 웹 크롤링 기능 개발

    • 다양한 웹사이트의 테이블 패턴을 분석하여 최대한 많은 페이지에서 크롤링이 가능하도록 구현

    • 페이지네이션 버튼을 감지하여 자동으로 페이지를 이동하며 여러 페이지를 크롤링할 수 있도록 개발

  • 데이터 필터링 기능 개발

    • 사용자가 원하는 키워드를 기준으로 포함(AND), 선택(OR), 제외(NOT) 조건을 설정할 수 있는 필터링 로직 구현

  • 핵심 로직의 단위 테스트 수행

    • 암·복호화 로직 모듈화 및 테스트

      • 단위 테스트가 가능하도록 암·복호화 로직을 독립적인 모듈로 분리

      • 다양한 데이터 구조를 테스트하여 중첩된 구조의 데이터도 정상적으로 암·복호화되는지 검증

    • 데이터 필터링 로직 테스트

      • 다양한 필터링 조건 조합에서도 필터링이 정상적으로 적용되는지 단위 테스트 수행
