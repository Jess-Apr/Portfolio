Site Scoop회사 프로젝트

프로젝트 소개

사용자가 지정한 웹 페이지의 테이블을 감지하여 동적 크롤링 후, 데이터를 엑셀 파일로 다운로드할 수 있게 해주는 크롬 확장 프로그램

  • 진행 기간: 2025.01 ~ 진행중
  • 진행 인원: 프론트엔드 1인, 백엔드 1인
  • 담당 업무: 프론트엔드 개발

사용한 기술 스택

TypeScriptReactJotaiTailwind CSSJest

주요 작업 내용

  • 클라이언트-서버 암호화 구현

    • 개인정보 필드를 AES로 암호화하고, AES 키를 RSA 암호화하여 개인정보 전송 보안 강화

    • 응답 데이터 암호화가 필요한 경우, 클라이언트에서 RSA 키를 생성하고 요청 헤더에 Public Key를 포함하여 전송

    • Axios Interceptor에 암·복호화 로직을 적용하여 모든 요청과 응답에 자동 암·복호화 처리 적용 (보일러플레이트 코드 최소화)

  • 웹 크롤링 기능 개발

    • 여러 웹사이트의 다양한 테이블 패턴을 분석하여 범용적인 크롤링 로직 구현

    • 페이지네이션 버튼을 감지하여 자동으로 페이지를 이동하며 여러 페이지를 크롤링할 수 있도록 개발

    • 복잡한 DOM 구조에서도 데이터를 정확히 추출할 수 있도록 DFS 기반 탐색 로직 구현

  • 조건 기반 데이터 필터링 로직 구현

    • 키워드 기반 조건식(AND / OR / NOT) 조합이 가능한 필터링 기능 개발

    • 필터 조합 시 예외 케이스를 고려한 로직 설계로 정확도 및 유연성 확보

  • 핵심 로직 모듈화 및 단위 테스트 수행

    • 암·복호화 로직과 필터링 로직을 독립 모듈로 분리하여 재사용성과 테스트 가능성 강화

    • 다양한 데이터 구조에 대해 단위 테스트 작성 → 데이터 무결성과 엣지 케이스에 대한 안정성 검증

문제 해결 경험

초기 화면 렌더링 최적화

  • 문제: RSA 키 생성 로직이 메인 스레드를 블로킹해, 초기 화면 렌더링이 지연됨

  • 해결: Web Worker를 활용해 RSA 키 생성을 비동기화하여 초기 화면 FCP를 평균 3.29s → 0.16s로 개선
